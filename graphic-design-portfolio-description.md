# Graphic Design Portfolio Carousel

This page presents a curated graphic design portfolio placement in a clean, modern carousel format. It highlights design thinking across branding, print, UI/UX, packaging, digital ads, and motion typography.

## Overview

The Graphic Design Portfolio Carousel is designed to showcase visual work in a polished, scrollable presentation. Each card describes a distinct concept or brief, from brand identity systems to campaign assets, helping visitors quickly understand the range of creative services offered.

## Key Features

- **Carousel layout** for easy browsing across multiple design concepts
- **Design-focused presentation** with clear tags, project titles, and brief descriptions
- **Visual hierarchy** that emphasizes branding, motion design, and portfolio placement
- **Subtle placeholder styling** for upcoming work, including a discreet "Coming Soon" treatment

## Highlighted Projects

- **Portfolio Placement** — a graphic design carousel page that showcases branding and UI concept explorations
- **Kinetic Typography** — motion design and typographic animation concepts for dynamic digital campaigns
- **Branding & Campaign Work** — identity and campaign visuals designed for premium brand experiences
- **Other concepts** include print layout, packaging, digital advertising, and UI/UX treatments

## Purpose

This page is intended as a visually engaging portfolio experience for prospective clients or employers in design-led roles. It reinforces the portfolio's creative direction while keeping the presentation consistent with the main web developer CV site.

## Technologies Used

- Responsive layout techniques
- Lightweight carousel design for smooth horizontal browsing
